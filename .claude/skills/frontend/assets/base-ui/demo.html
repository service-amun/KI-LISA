Convex (card) vs concave (card-concave) — same depth, opposite direction

Convex surface

Light hits top-left, shadow falls bottom-right. Element protrudes from the page — use for interactive surfaces, cards, buttons.

panel elevated

Concave surface

Shadow falls top-left, light comes bottom-right. Element recesses into the page — use for wells, input areas, progress tracks, sidebar panels.

well recessed

Concave — nested inside convex card

An input well, search area, or inset data panel. The outer card protrudes; the inner concave sinks — layered depth without a flat surface in sight.
