# Proper-Noun Accuracy

Reference for P2 (Names and proper nouns). Tech trademarks and product names must be
capitalized exactly per their owner's official branding. Flag any deviation; in a
customer-facing deliverable a wrong proper noun is at least `major`, a wrong client or
person name is `blocking`.

| Correct | Incorrect forms |
|---|---|
| JavaScript | Javascript, javascript |
| TypeScript | Typescript, typescript |
| GitHub | Github, github |
| GitLab | Gitlab, gitlab |
| npm | NPM (except in headings where all-caps is conventional) |
| Node.js | NodeJS, Node, node.js |
| React | react |
| macOS | MacOS, macos, Mac OS |
| iOS | IOS, Ios |
| PowerShell | Powershell, powershell |
| Claude | CLAUDE (when referring to the model or product) |
| Claude Code | ClaudeCode, claude-code |
| Anthropic | anthropic |
| OpenAI | Openai, openai, Open AI |
| OWASP | Owasp, owasp |
| NIST | Nist, nist |
| TOON | Toon, toon (when referring to the format) |

For client, partner, and person names: there is no master list — verify each against the
source brief or prior correspondence, and confirm the spelling is identical at every
occurrence in the deliverable.
